What's inside

Component
Specs

Laravel Tinker Mode

Auto-detects Laravel projects via the artisan file. Bootstraps the full application context so your code runs inside the framework, not beside it.

Monaco Code Editor

Powered by the same engine as VS Code. Full PHP syntax highlighting, bracket matching, and multi-cursor editing built in from day one.

SQL Runner

Write and execute raw SQL queries against your project's database connection. Results render in a reactive, sortable table view.

Live Log Viewer

Real-time tail of storage/logs/laravel.log. Streaming updates as your application writes them, without refreshing.

SSH & Docker

Connect to remote PHP environments over SSH or run code inside Docker containers. The exact same editor interface for local and remote execution.

Hybrid IntelliSense

LSP-driven Eloquent model autocompletion backed by a Rust symbol indexer. Completions that know your schema, not just the PHP spec.
